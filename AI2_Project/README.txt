This package consists of 5 files
BullshitGame.py
	Contains game logic
Cards.py
	Class for card objects
Player.py
	Outer class wrapper for human or ai players
PlayerType.py
	Contains AI logic and human payer type
gamePlay.py
	Interactive game
gameRun.py
	Simulation

To run AI bullshit you have two options.

1. Run simulations
	$ python3 gameRun.py

2. Play a text based game against 4 AIs
	$ python3 gamePlay.py

NOTE --- this game requires python 3 and WILL NOT run on an older version of python.
